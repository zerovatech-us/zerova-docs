Upgrade​

All of firmware image can be upload from this page to upgrade, charger will auto check image header & CRC32 before upgrade.

If image valid pass, charger will upgrade in IDLE mode and reboot after upgraded.

Charger indicator in upgrading please reference each model system specification.
